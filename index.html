Illustrera

Float introducerades i Netscape 1.1 år 2002 med syftet att "flyta" en text runt ett annat element, t.ex. en bild.

Float i samverkan med div-boxar fungerade dock bättre för layout än de tidigare verktygen när man använde tables som var ännu krångligare. Float var aldrig tänkt att användas för sidlayout men det var nu det bästa verktyget som fanns tillgängligt.

Problemet med float när det används för sidlayout är att man måste använda sig av extra markup i html och extra kod i CSS för att få saker att hamna där man vill. Clearfix behövs för att bryta floateffekten nedåt på sidan så att inget content kryper uppåt. En clearfix är en tom div med CSS-stylingen "clear-both". Clearfix är bara ett av många så kallade "hacks" som användes för att kunna använda float som layoutverktyg.
